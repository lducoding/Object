객체는 기능을 구현하기 위해 협력하는 일원

---- 먼저 기능을 정의해보자 ----

걸음수로 랭킹을 정하려고 한다.
참여자들의 걸음 정보가 입력이 되면 최고로 많이 걸은 사람 1,2,3 등이 나온다.

---- 기능을 특정 해보자 ----
사용자별 걸음수 저장
많이 걸은 순서로 1,2,3 저장해 놓기

---- 협력은 어떤 객체가 다른 객체에게 무언가를 요구하는 것 ----
---- 협력을 조금 더 자세하게 풀어서 정의 해보자 ----

사용자가 걷는다.

걸음의 데이터가 축적된다.

00시가 되면 걸음정보가 사용자 이름과 함께 저장된다.

-------

랭킹안에는 사람과 걸음수가 연결되어 있어

그 안에서 걸음수로 정렬하면 된다.

---- 협력에 따른 객체 정의 ----
사용자 걷는다.
User 객체는 walk를 상태로 가진다.
         name 을 상태로 가진다.
